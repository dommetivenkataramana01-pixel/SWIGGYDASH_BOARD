# Swiggy Sales Dashboard 📊

## 🎯 Objective

To analyze Swiggy sales data and build an interactive dashboard that provides meaningful business insights and helps track overall sales performance.

## 📂 Raw Data

The project contains raw sales data, including:

- Restaurant details
- Order information
- Food categories
- Sales figures
- Customer ratings
- State-wise data

## 📌 KPIs

- 💰 Total Sales: ₹6.83M
- ⭐ Average Rating: 4.34
- 🛒 Average Order Value: ₹268.73
- 📦 Total Orders: 25.40K
- 📊 Ratings Count: 0.72M

## 📈 Visualizations

- Monthly Sales Trend Analysis
- Daily Sales Trend Analysis
- Food Type Sales Distribution
- State-wise Sales Analysis
- Top 5 Cities by Sales
- Quarterly Sales, Ratings, and Orders Analysis
- Interactive Filters (Months, Restaurant Name, Category)

## 💡 Business Insights

- Identified top-performing cities and states.
- Analyzed monthly and daily sales trends.
- Compared sales distribution across food categories.
- Evaluated customer ratings and order patterns.
- Tracked overall business performance using KPIs.

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Analysis
- Data Visualization
