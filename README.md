Swiggy Sales Dashboard 📊

This project is an interactive Excel dashboard created to analyze Swiggy sales data and derive meaningful business insights.

Features
📈 Monthly and daily sales trend analysis
🍽️ Food category-wise sales distribution
🗺️ State-wise sales analysis
🏙️ Top-performing cities by sales
⭐ Average customer rating analysis
💰 Average order value tracking
🎛️ Interactive filters for months, restaurants, and categories
Tools & Skills Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
KPI Analysis
Data Visualization
Objective

Transform raw sales data into an interactive dashboard that helps identify trends, monitor performance, and support data-driven decision-making.
